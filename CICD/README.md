# 持续集成进阶
---

## 闲谈：

- 目标一致性：
    - 加快开发迭代速度
    - 较少人工成本
    - 快速定位问题
    
    
- 依赖模块：
    - CMDB（资产管理系统）
    - CM（配置管理）
    - GIT/SVN（版本控制）
    - PackageManager（包仓库管理）
    - CI (持续集成，自动化测试待学习)
    - CD（持续发布）
